Batch modify the translate independently settings of WPML for all or selected posts of a given language.

== Description ==

This plugin works with WPML and provides the capability of batch modifying the translate independently settings of all or selected posts of a given language. It has been built so that it uses zero resources outside its dedicated admin panel page. 

For a preview of what the plugin does and how it can be used, please have a look in the screenshots section. 

Requirements:

- Active WPML installation is a prerequisite for the plugin to work.

- Tested OK for WPML 3.3 and above (may also work for earlier WPML versions but it has not been tested for that).

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Click on Settings->Batch Translate Independently to use.
